CSS

Cascading Style Sheet
태그들의 속성값에 파싱하여 디자인을 적용시키는 파일

selecter

tag, class, id 의 값에 파싱하여 프로퍼티를 적용한다. attribute에 따라 우선순위가 다르다.

  1. Tag attribute

    2. 태그로

  2. class attribute

    3. class로

  3. id

    4. id로

    "!import"로 우선순위를 강제로 정할 수도 있다.

selecter property

  • A { } : 기본적인 방법
  • A > B { } : A의 바로 직속자식인 B를 가지고 있는 모든 태그
  • A, B { } : A와 B 둘중에 하나라도 가지고 있는 태그
  • A B:nth-child(?) { } : A의 자식인 B를 가지고 있는 태그중 ?번째
  • A B: first-child { } : A 자식인 B를 가지고 있는 태그중 첫번째(마지막은 last)
  • A B:not (:first-child) { } : A자식중 B를 가지고 있는 태그중 첫번째만 빼고 모두(마지막은 last)
  • A : hover { } : 마우스가 A에 올라갔을때
